# Kogado Studio

Extracts HyPack archives used in visual novels by Kogado Studio.

## Games Tested

* [Hakuisei Ren'ai Shoukougun](http://www.kogado.com/sw/contents/shimarisu/shirokoi/)
* [Hakuisei Aijou Izonshou](http://hakuai.kogado.com/)
